## Відомі обмеження
Оскільки даний проєкт є прототипом (MVP) для наукового дослідження, існують певні обмеження:
1. Дані емулюються локально і скидаються при перезавантаженні сторінки (відсутність бекенду).
2. Окремі функції (наприклад, сторінка налаштувань) знаходяться в розробці.
3. Можливі візуальні артефакти на екранах з нестандартною роздільною здатністю.

Ці аспекти будуть виправлені на етапі підготовки до пілотного впровадження.

## Особливості навігації прототипом
1. Для демонстрації "синхронізації" не перезавантажуйте оцінку.
2. Для переходу між ролями натисніть зображення профіля
   (сіреньке коло в панелі інструментів на ПК зліва зверху, на телефоні зправа зверху)
3. Щоб побачити результат виставлення оцінки в кабінеті учня, її треба давати Новицькому Володимиру
